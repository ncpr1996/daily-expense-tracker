## Troubleshooting

### Data Not Saving
- Check if browser allows localStorage
- Try different browser
- Clear browser cache and retry

### Chart Not Showing
- Add some expenses first
- Select a month with expenses
- Refresh the page

### Budget Not Updating
- Make sure you clicked save
- Check if expenses are in current month
- Refresh the page

### Dark Mode Not Working
- Clear browser cache
- Check browser compatibility
- Try toggling multiple times

## Frequently Asked Questions

**Q: Can I use this offline?**
A: Yes! Once loaded, it works offline.

**Q: Is my data synced across devices?**
A: No, data is stored locally per browser.

**Q: Can I export my data?**
A: Currently no, but you can take screenshots or manually copy data.

**Q: What happens if I clear browser data?**
A: All expense data will be lost. Backup important data first.

**Q: Can multiple people use this?**
A: Yes, but data will be mixed. Use different browser profiles.

**Q: Is there a mobile app?**
A: No, but the web app works great on mobile browsers.

**Q: Can I customize categories?**
A: Currently no, but you can use "Other" for custom needs.

**Q: What's the maximum number of expenses?**
A: Depends on browser storage (~5-10MB), typically 1000+ expenses.
